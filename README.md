# Algoprog Problems

This repository contains my solutions to algorithmic problems from the Algoprog course (https://algoprog.org).

## 📌 About

Algoprog is an online course focused on algorithms and data structures, where problems are automatically tested and reviewed for correctness and code quality.  
Through this repository, I practiced solving a wide range of algorithmic challenges and improving problem-solving skills.

## 🧠 Topics Covered

The solutions in this repository include problems related to:

- Sorting and searching
- Graph algorithms
- Dynamic programming
- Greedy algorithms
- Data structures
- Basic number theory

## 🛠️ Technologies

- Python (main language), C++

## 🎯 Goals

- Strengthen algorithmic thinking
- Write clean, efficient, and reusable code
- Practice solving problems under constraints
- Prepare for technical interviews and competitive programming

## 📁 Structure

The repository is organized by topics and difficulty levels (following the Algoprog structure).

## ⚠️ Notes

- Solutions are written for learning purposes and may not always represent the most optimal approach
- Focus is placed on clarity and correctness, with gradual improvements over time
